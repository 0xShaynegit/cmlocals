## Overview

This document outlines the approved design for the CMLocals homepage restructure, implementing Requirements v3 with a focus on content hierarchy, visa option prioritization, and improved user navigation flow.

---

## Core Design Principles

1. **Navigation** – Exact match to requirements menu structure
2. **Content Hierarchy** – Education, Volunteer, DTV, LTR positioned as Priority 1 throughout; Retirement/Marriage clearly secondary with disclaimers
3. **Section Sequence** – Complete restructure to follow linear visa decision flow
4. **Getting Settled** – New flexible card section with dynamic layout (text not always left-aligned)
5. **Hero Content** – Update to match requirements v3 exactly
6. **Footer** – Clean structure without change log; organized by topic columns
